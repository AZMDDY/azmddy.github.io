# Mac(M2) 安装conda

Conda是一个辅助的包管理和环境管理的工具

从清华源下载：Index of /anaconda/archive/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror (opens new window)

从官网下载：Free Download | Anaconda (opens new window)

两者都可以下载，可以下载带图像界面的包，也可以下载命令行安装的包。

我这安装的是图形界面。安装的结果如下：

接下来就可以使用conda创建项目啦！

# 创建名称为test的项目，python版本为3.11
conda create -n test python=3.11

# 激活环境,激活后方可使用
conda activate test
