打印数组下标

# 打印数组下标

默认情况下，gdb 打印数组时不打印下标。

#include <iostream>
using namespace std;
int main(int argc, char** argv)
{
    int a[] = {9, 4, 5, 3, 5, 6, 7};
    cout << a[0] << endl;
    return 0;
}
