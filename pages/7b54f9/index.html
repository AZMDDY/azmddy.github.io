编译期分派

# 编译期分派

“分派”即指根据条件执行不同的语句。对于执行期进行分派，if-else或switch就可以了。对于编译期分派，除了使用条件编译之外，还可以利用模板的方式实现。

实现方式是将常整数映射为类型并通过重载进行编译期分派。

template <int v>
struct Int2Type
{
    enum { val = v };
};

Int2Type会根据模板参数的不同产生不同的类型，即Int2Type<1>不同于Int2Type<2>，这一点是在编译期就确定好的。

编译器并不会去编译一个未被使用到的模板函数，编译器只是对它进行文法检查。

template<int v>
struct Int2Type {
    enum { val = v };
};

class Test {
public:
    void DoSomething(int arg)
    {
        DoSomething(Int2Type<true>(), arg);
    }

private:
    void DoSomething(Int2Type<true>, int arg) {}
    void DoSomething(Int2Type<false>, int arg) {}
};
