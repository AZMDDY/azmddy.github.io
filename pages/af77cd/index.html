# HPE gen10 plus 安装ESXI 7

# ESXI下载地址

VMware ESXi Images for HPE Servers | HPE Taiwan (opens new window)

# 安装

将一个空U盘插到内部的USB口上。

打开终端控制台

导入已经下载好的ESXI的ISO文件

选择一次性引导项

重启

按住F11 ，进入“一次性启动菜单”，然后选择iLO Virtual USB3: iLO Virtual CD-ROM

等待安装，注意选择安装位置到你的U盘：
