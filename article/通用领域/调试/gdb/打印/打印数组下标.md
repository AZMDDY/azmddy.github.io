# 打印数组下标

默认情况下，gdb打印数组时不打印下标。

```cpp
#include <iostream>
using namespace std;
int main(int argc, char** argv)
{
    int a[] = {9, 4, 5, 3, 5, 6, 7};
    cout << a[0] << endl;
    return 0;
}
```

![image-20220301211845411](https://cdn.jsdelivr.net/gh/AZMDDY/imgs/img/image-20220301211845411.png)

如果要打印下标，可以使用`set print array-indexes on`命令，这就能简单知道下标对应的值了。
