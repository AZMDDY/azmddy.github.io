# 代理设置

【环境】：Linux, Go1.13

设置`GOPROXY`代理：

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

设置`GOPRIVATE`跳过私有库

```bash
go env -w GOPRIVATE=*.gitlab.com,*.gitee.com
```

设置`GOSUMDB`提供sum验证服务，验证包的有效性。Go1.13默认的GOSUMDB=sum.golang.org，这个访问不了，可以改为以下内容:

```bash
go env -w GOSUMDB="sum.golang.google.cn"
```
